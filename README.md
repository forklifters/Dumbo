![Dumbo](./resources/dumbo.jpg)

## Development

```sh
yarn run start:webpack
```

```sh
yarn run start:electron
```

## Configuration

Configure Dumbo with `~/.dumbo/dumborc.json`.

### `Theme`

You can choose a theme `WHITE` or `BLACK`. If configuration file does not exists, set to `WHITE`.

```json
{
  "theme": "WHITE"
}
```
